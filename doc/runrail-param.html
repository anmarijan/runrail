2 speedtraction

2.1 Summary

obejct name type unit description
id INT - Unique Number
name STRING - Name
labels ARRAY - Label of x-axis and y-axis
units ARRAY - Unit of x-axis and y-axis
data ARRAY - Array of speed-traction pair

2.2 Remarks

2.2.1 id

This is refered from train data as "sptrindex".

2.2.2 units

The unit of speed and traction power.

  • speed: Not used in this version
  • traction power: "kgf" and "N"

3 train

3.1 Summary

obejct name type unit description
id INT - Unique Number
name STRING - Name
maxspeed REAL km/h Maximum speed
jerk REAL m/s3 Jerk(not used)
length REAL m Length
WT REAL ton Total weight of trailer cars
WM REAL ton Total weight of motor cars
nCars INT - The number of cars per train
traction STRING - Traction data type
motorindex INT - Index of the motor
sptrindex INT - Index of the speed-traction relationship
lineindex INT - Index of the line
nTractions INT - The number of traction units
resistance OBJECT - Information of rolling resistance
model STRING - Model name of the rolling resistance
params ARRAY - Parameters of the model above

3.2 Remarks

3.2.1 traction

If traction is "motor", the motor model is used.

3.2.2 resistance/model

In the table, "units" means those of variables such as speed, weight, and output value.

Name units No. of parameters description
None - 0 Rolling resistance is 0. Stating resistance is calculated.
Quadratic km/h, -, N 3 T=a+b*v+c*v2
JNR_EMU_MT km/h, ton, kgf 6 A popular method in Japan. Different parameters for motor cars and trailer cars
JNR_EMU km/h, ton, kgf 4 A popular method in Japan.

4 Motor

4.1 Summary

object name type unit description
id INT - Unique Number
power REAL kW Max power of the motor
volt REAL V Voltage
pole INT - No. of poles
efficiency1 REAL - Power efficiency rate at point 1
efficiency2 REAL - Power efficiency rate at point 2
freq1 REAL - Input frequency at point 1
freq2 REAL - Input frequency at point 2
fslip1 REAL - Slip frequency at point 1
fslip2 REAL - Slip frequency at point 2
maxspeed REAL km/h Max train speed
diameter REAL mm Diameter of a wheel
gear REAL - Gear ratio

4.2 Remarks

The model needs parameters of efficicency, frequency, and slip at two points. The first one is the change point between the constant torque region and constatnt power region of the induction motor. The second one is the change point between the constant power region and breakdown torque region.
